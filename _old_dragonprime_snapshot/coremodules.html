Author: Talisman and Chris Vorndran

This list provides a basic description of all core modules included in version 1.0.6. I have not gone into instructions for configuring or using these modules, nor have I provided complete description down to every outcome, etc. If you feel I've missed any important points with these modules, please IM Talisman or Sichae. Note that 'required' modules are all core files included with the original game archive. Modules are listed alphabetically by FILENAME, not by module name.

Abigail the Street Hawker [Top]
Filename: abigail.php
Category: Village Special Event
Author: Shannon Brown (Saucywench)
Required Modules: None
Short Description: Allows you to exchange gems for jewelry for your lover. This may bring you some extra charm on the next newday.
Add Gems Module [Top]
Filename: addgems.php
Category: Village
Author: Shannon Brown
Required Modules: None
Short Description: While active, gives every user who logs in 2 gems “As our way of saying thank you for all the wonderful donations you have pledged to Child's Play during November, we are pleased to present every player with two gems. Thanks for your support!” This gift is only granted once to each player.
